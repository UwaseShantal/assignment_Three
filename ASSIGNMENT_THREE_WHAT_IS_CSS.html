WHAT IS CSS3

CSS3 stands for Cascading Styling Sheet 3 is a more advanced version of CSS that is used for representing the look and design of an HTML document

CSS3 is important in web development because it helps to define the layout of web pages, making them better looking and more user-friendly



Seperation of concerns in web development

Separation of concerns is the idea that computer programs should be seperated into distinct sections that work with asingle feature.



Selectors in CSS

These are patterns of elements and other terms that tell a browser which HTML elements should be selected to have the CSS property values inside the rule appliedto them.

Some types of CSS selectors are element selectors, class rules, attribute selectors, pseudo-class selectors and psuedo element selectors
